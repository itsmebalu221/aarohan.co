/* ═══════════════════════════════════════════════════════════
   FONT CONFIGURATION
   ═══════════════════════════════════════════════════════════
   
   This project uses PP Neue Montreal as the primary typeface.
   
   To use the fonts:
   1. Obtain PP Neue Montreal from Pangram Pangram Foundry
   2. Place the following files in /public/fonts/:
      - PPNeueMontreal-Book.woff2
      - PPNeueMontreal-Medium.woff2
      - PPNeueMontreal-Bold.woff2
   
   Alternative free fonts that maintain the design intent:
   - Inter (similar geometric sans)
   - Satoshi (by Fontshare)
   - Outfit (Google Fonts)
   
   The font system is designed with these characteristics:
   - Display: Medium/Bold weight for headlines
   - Body: Book/Regular weight for paragraphs
   - Tight tracking for display text
   - Slightly loose tracking for small labels
   
   ═══════════════════════════════════════════════════════════ */
